下载与安装

如何下载并安装快连VPN？

前往 下载页面，选择你的系统（Windows、macOS、Android、iOS、Linux）并获取安装包。安装后登录或注册新账号，选择就近的高速节点，一键连接即可使用。

Windows 安装失败怎么办？

请按以下顺序排查：

  • 以管理员身份运行安装程序。
  • 关闭安全软件后重试，或将安装目录加入白名单。
  • 确保系统为 Windows 10/11 64 位，并更新 .NET 运行库。

提示：部分旧版系统可能缺少驱动签名与虚拟网卡依赖，更新系统后再试。

iOS 与 Android 如何安装？

iOS：从 App Store 搜索「快连VPN」或访问 iOS 下载。Android：可通过 Google Play 安装或直接下载 APK（本地 APK）。

Linux 是否支持？

支持主流发行版（Debian/Ubuntu/CentOS 等），提供 CLI 与 GUI 两种方式，详见 Linux 安装指引。

账号与订阅

是否必须注册账号？

建议注册账号以获得完整功能与设备同步。部分平台支持匿名试用，但到期后需绑定邮箱以继续使用。

订阅包含什么？是否支持多设备？

订阅解锁全球节点、智能选点、专线加速与多协议等权益。一个账号可在多台设备同时登录，具体并发数量以套餐为准（可在「设备管理」中查看）。

如何变更套餐或续费？

在客户端「账户/订阅」中选择月度、季度或年度方案完成支付即可。来自 App Store / Google Play 的订阅需在各自商店管理。

是否提供发票？

支持企业抬头与电子发票，请在支付后通过 联系我们 提供抬头、邮箱与订单号。

连接与速度

速度慢/不稳定怎么办？

先尝试更换就近节点，并在设置中切换协议（UDP/TCP/Socks5）。高峰时段可使用智能选点或优先选择负载较低的服务器。

怎么选择最合适的节点？

使用客户端的「一键测速」或「自动选点」功能，综合 延迟、丢包、负载 指标选择；跨区访问建议就近国家/地区的中转节点。

支持端口转发吗？

专业版可开通端口转发（仅白名单端口），用于远程桌面、开发调试等场景。请联系技术支持开通并获取使用指引。

是否支持分应用/分流？

支持。你可以把指定站点或应用加入「直连/代理」列表，常见于网银、政务或内网访问的兼容需求。

隐私与日志

快连VPN 是否记录上网数据？

我们遵循「不记录、不追踪、不出售」原则，不保存访问内容、原始 IP 或可识别的行为日志。仅在必要时采集匿名性能指标用于网络优化，详见 隐私政策。

会共享个人信息给第三方吗？

不会。除非依法合规的强制要求，为维护服务安全或用户合法权益，才可能在最小化原则下配合。

支持哪些加密协议？

支持多种 VPN 协议（如 UDP/TCP/Socks5 等），内置混淆与抗干扰策略，保障连接可靠性与私密性。

跨平台使用

一个账号可以在几台设备上同时使用？

默认支持多设备并发（电脑与手机同时在线）。若需更多并发，请升级更高套餐或联系商务开通。

是否支持路由器？

可通过支持代理的路由器或软路由接入，让全家设备共享加速。请参考「路由器配置指南」或咨询技术支持。

企业如何集中管理成员？

企业版提供成员邀请、统一结算与访问控制策略。支持按项目/部门维度下发节点与分流规则。

进阶设置与网络环境

如何开启开机自启、自动重连与 Kill-Switch？

在客户端「设置」中打开对应开关即可。Kill-Switch 会在网络异常时阻断流量外泄，保障隐私安全。

在受限网络/校园/酒店Wi-Fi下如何使用？

启用「智能穿透」或更换为抗干扰协议，必要时手动指定端口。仍受限时建议改用移动网络或咨询本地管理员。

可以自定义 DNS 或启用广告拦截吗？

在「网络设置」中自定义 DNS，推荐 DoH/DoT 方案以获得更好的隐私与解析成功率。广告拦截可在「安全」页启用。

常见故障排查

能连上但打不开部分网站？

清理浏览器缓存与 DNS 缓存（ipconfig /flushdns），更换协议或换到其他节点，再测试。若是目标网站风控导致，请启用专属 IP。

总是掉线或频繁重连？

检查本地网络质量（Wi-Fi 干扰/弱信号），切换到 5GHz 或有线；在「设置」中开启自动重连与网络守护。

应用商店下载不了/更新失败？

请切换到目标地区的节点后重试；如仍失败，手动下载安装包或使用备用商店。
